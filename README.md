
# Flujo usar git

Agregar archivos
```bash
  git add .
```

Poner un mensaje al commit
```bash
  git commit -m "mensaje aqui"  
```

Subir archivos
```bash
  git push
```

## Compilar programas
```bash
  g++ -std=c++17 nombre_programa.cpp -o nombre_programa
```

## Ejecutar programa pasandole un archivo
```bash
  ./nombre_programa < archivo.txt
```
